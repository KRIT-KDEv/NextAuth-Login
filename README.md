Secure Authentication System (Next.js + NextAuth.js)
A robust and secure user authentication and registration system built with Next.js and NextAuth.js, featuring stateless session management and secure data persistence in MongoDB.

Framework: Next.js (App Router)

Database: MongoDB with Mongoose for user profile storage 
Security: JWT (JSON Web Tokens) & Secure Password Hashing 
Styling: Tailwind CSS for a responsive UI 

Key Features
Stateless Authentication: Implemented JWT (JSON Web Tokens) to manage secure session tokens, ensuring scalable and stateless communication between client and server.
Secure Credential Storage: Integrated MongoDB as the primary database to store user profiles and securely hashed credentials.
Protected Routes: Leveraged Next.js Middleware to restrict access to sensitive pages, ensuring only authenticated users can access specific resources.
Flexible Auth Providers: Configured NextAuth.js to handle secure user registration and login workflows efficiently.
Responsive Auth UI: Designed a clean, user-friendly authentication interface using Tailwind CSS.
